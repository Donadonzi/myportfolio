Mentorshare App

During the Hatchways co-op program, I collaborated with a team in an Agile environment using ZenHub for ticket management, to build a web application that allows users to upload their code snippets and have them reviewed by another developer, utilizing MERN stack, Material-UI and Stripe. I Implemented authentication flow using JWT and cookies; Developed components, models and API routes for creating and updating user profile and uploading code; Implemented avatar uploading feature using multer and sharp libraries; Utilized Socket.io to generate real-time notifications and implemented reviewer matching algorithm.

Surveyor App

An application for sending out surveys and collecting back data, built with Node/Express, React/Redux/Redux Form and MongoDB, deployed on Heroku. Login feature is implemented using Google OAuth and Passport.js. Handling payments with React Stripe Checkout and Stripe Node.js, it allows users to send customized surveys to their own recipients. It uses SendGrid and its event webhook for sending emails and click tracking.
